# RPS-The-Game
### Local Multiplayer Game using tKinter

## Prerequisite:-
Python 3

### Required Libraries:-

1) Pillow
2) tKinter

## Instructions to play:-
* run server.py file in terminal.
* run Main.py file in second terminal.
* run Main.py file in third terminal.
